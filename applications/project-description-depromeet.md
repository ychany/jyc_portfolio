## 답변 (줄글, 약 490자)

디프만 18기에서 진행한 위시리스트 소비 결정 서비스는 사고 싶은 것은 많지만 선택이 어려워 구매를 미루는 사용자를 위해, 쌓인 위시리스트에서 '먼저 살 것'을 골라주는 서비스입니다. 저는 Next.js 기반 웹 프론트엔드를 맡았고, 기획 단계부터 참여해 서비스의 핵심 기능을 어떻게 사용자 경험으로 풀어낼지 팀과 함께 고민했습니다. 그 결과 두 상품을 하나씩 비교해 최종 1순위를 정하는 '1:1 토너먼트' 기능을 핵심 재미 요소로 제안하고, 진입부터 비교 진행, 결과를 영수증 형태로 보여주는 화면까지 전 과정을 직접 설계·구현했습니다. 이때 라운드가 빠르게 넘어가며 결과가 꼬이던 문제를 상태 처리 로직을 다듬어 해결했고, 이후 친구와 함께 고르는 실시간 소셜 토너먼트로 기능을 확장했습니다. 또 로그인이 갑자기 풀리던 인증 문제를 잡고, 사용자 행동 데이터를 분석해 이탈 지점을 개선하는 작업도 맡았습니다. 화면을 만드는 데 그치지 않고 실제 사용 흐름에서 불편한 부분을 끝까지 파고들어 다듬었으며, 팀 공통 개발 규칙을 정리해 협업이 원활하도록 기여했습니다. 이 프로젝트는 최종 발표에서 **대상**을 수상했습니다.
